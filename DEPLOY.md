# Инструкция по деплою на Vercel

## Важная информация о LocalStorage

⚠️ **Важно понимать:**
- LocalStorage работает в браузере и сохраняет данные **локально на каждом устройстве**
- Данные **НЕ синхронизируются** между устройствами
- Если вы добавите микс на телефоне, он не появится на компьютере
- Данные хранятся в браузере конкретного устройства

## Для синхронизации между устройствами нужно:
- Использовать Firebase Firestore
- Или Supabase
- Или другую облачную базу данных

## Шаги для деплоя:

### 1. Закоммитить изменения в Git

```bash
git add .
git commit -m "Initial commit: Hookah Mix Creator app"
```

### 2. Создать репозиторий на GitHub

1. Зайдите на https://github.com
2. Нажмите "New repository"
3. Назовите репозиторий (например: `hok-mix`)
4. НЕ добавляйте README, .gitignore или лицензию (они уже есть)
5. Нажмите "Create repository"

### 3. Подключить локальный репозиторий к GitHub

```bash
git remote add origin https://github.com/ВАШ_USERNAME/hok-mix.git
git branch -M main
git push -u origin main
```

### 4. Деплой на Vercel

1. Зайдите на https://vercel.com
2. Войдите через GitHub аккаунт
3. Нажмите "Add New Project"
4. Выберите ваш репозиторий `hok-mix`
5. Vercel автоматически определит Next.js
6. Нажмите "Deploy"

### 5. Готово!

Vercel автоматически:
- Соберет проект
- Задеплоит его
- Дам вам URL (например: `hok-mix.vercel.app`)
- При каждом push в GitHub будет автоматически обновляться

## Доступ с телефона

После деплоя вы получите URL вида:
- `https://hok-mix.vercel.app`
- Или кастомный домен, если настроите

Просто откройте этот URL на телефоне в браузере!

## Что будет работать:

✅ Все функции приложения
✅ LocalStorage (данные сохраняются локально на каждом устройстве)
✅ Добавление/редактирование/удаление миксов

❌ Синхронизация данных между телефоном и компьютером
❌ Общие миксы между разными браузерами/устройствами

## Для синхронизации (будущее):

Если захотите синхронизацию, можно добавить:
- Firebase Firestore (бесплатный план до 1GB)
- Supabase (бесплатный план)
- Или другую БД
